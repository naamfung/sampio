##SAMPIO 三拼粤语输入法

用「juo」法「fat」：

1. sam pio——三拼
2. jjt jj——粤语
3. sj jbp fat——输入法
4. 此为「wbi」基于「jj」粵拼之简「gan」拼
5. 每字不带调「diu」不超过「uh」三键「gin」，带「dai」调不超过四键
6. 使用26只「zgk」英文字母输入
